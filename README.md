# UniDive Winter School — Hands-on Sessions (LLMs & Low-Resource NLP)


This folder contains the **hands-on notebooks** for the UniDive Winter School training on **low-resource NLP and LLMs**. The sessions move from setup and multilingual analysis to dialogue summarization, prompt engineering, fine-tuning, and bias auditing.

## Trainers

[Nina Hosseini-Kivanani](https://www.linkedin.com/in/ninahkivanani) (RTL & University of Luxembourg)  
[Dimitra Anastasiou](https://www.linkedin.com/in/dimitraanastasiou/) (Luxembourg Institute of Science and Technology (LIST))  

## Sessions

1. **Session 0:** Orientation & setup  
   `Session0_Orientation_and_Setup_LLMs_Low_Resource.ipynb`
2. **Session 1:** Dialogue Summarization (low-resource baselines)  
   `Session1_Dialogue_Summarization_Low_Resource.ipynb`
3. **Session 2:** Prompt Engineering  
   `Session2_prompt_engineering.ipynb`
4. **Session 3:** Fine-tuning LLMs for Low-Resource Languages  
   `Session3_Fine_tuning_LLMs_for_Low_Resource.ipynb`
5. **Session 4:** Bias Audit  
   `Session4_Bias_Audit.ipynb`

## What You Will Learn

- Multilingual tokenization and embedding analysis
- Dialogue summarization with TextRank and extractive methods
- Prompt engineering patterns and evaluation
- Fine-tuning strategies for low-resource languages
- Bias detection and mitigation in NLP systems

## How to Use the Materials

You can run the notebooks in **Google Colab** or locally:

- **Colab (recommended):** open the notebook and enable GPU in `Runtime → Change runtime type → GPU`
- **Local Jupyter:** install dependencies as instructed inside each notebook

## Prerequisites

- A laptop with internet access  
- A Google account (for Colab) or a local Python 3 environment  
- Basic familiarity with Python and Jupyter notebooks
